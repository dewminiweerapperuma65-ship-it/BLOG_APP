# Blog App

A simple PHP + MySQL blogging application where users can:
- Register, login, and manage accounts
- Create, edit, and delete blog posts
- Add and delete comments

## Tech Stack
- PHP (Backend)
- MySQL (Database)
- HTML, CSS, JavaScript (Frontend)

## Live Demo
[https://learnblog.infinityfreeapp.com](https://learnblog.infinityfreeapp.com)

## Setup
1. Import the SQL database into phpMyAdmin.
2. Update `db_connect.php` with your DB credentials.
3. Host on any PHP-supported server (like InfinityFree or XAMPP).
